Ye Old Newspaper
Class project developed with a teammate that tasked us with the use of two seperate APIs working together to create something. Utilizing newsapi.org & Funtranslations API, we created a news site that converted all news articles to old english/shakesperean language. View on Github
Libraries & Apis
  • Newsapi.org
  • Funtranslations
Features
  • Select different topics
  • View articles in old english
  • Select articles to read the full story
Accomplished
  • Utilized two different APIs to work together
  • Collaborated with a teammate by splitting and delegating work for one another
  • Produced a working version that could be judged by the professor
